# Bundesliga Forwards Statistics Database
A SQL database containing statistical data for 6 forwards in the German Bundesliga in the 2022/2023 season.

## Features
- Store and retrieve statistical data for 6 forwards.
- Query data to analyze player performance.
- Easy-to-understand database schema.

## Usage
You can use this database to perform various operations, such as:
- Retrieve a forward's goal statistics for a specific season.
- Compare the performance of multiple forwards.
